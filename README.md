# Single-node-kubernetes
Single node Kubernetes cluster with Docker runtime &amp; build through kubeadm

## How to use
Requirements:
```
Vagrant 2.2.7
VirtualBox 6.1
```
Command to roll out single node Kubernetes:
`vagrant up`
